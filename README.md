## Install
1. Clone the repo
~~~
git clone https://github.com/vs74/EEG
cd EEG
~~~
2. Set up Virtual Environment
```
virtualenv eeg_env --python=3.6
source eeg_env/bin/activate
```

3. Now, in the terminal run
```
bash install_files.sh
```
